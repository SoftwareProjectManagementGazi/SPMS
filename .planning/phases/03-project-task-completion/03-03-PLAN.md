Note: `GET /sprints?project_id=X` means the `project_id` comes as a query parameter, not path param. The `get_project_member` dependency expects `project_id` as a path param — create a thin `get_sprint_project_member` dependency that reads `project_id` from query param instead:
```python
async def get_sprint_project_member(
    project_id: int,  # query param from /sprints?project_id=X
    current_user: User = Depends(get_current_user),
    project_repo = Depends(get_project_repo),
) -> User:
    if _is_admin(current_user): return current_user
    project = await project_repo.get_by_id_and_user(project_id, current_user.id)
    if not project:
        raise HTTPException(403, "Access denied")
    return current_user
```
