**comment_repo.py** — `SqlAlchemyCommentRepository`:

`_get_base_query()` always filters `CommentModel.is_deleted == False` (critical — per Pitfall 7 in research). Joins `UserModel` via `joinedload(CommentModel.user).joinedload(UserModel.role)` so author info is available.

Methods:
- `get_by_id(comment_id)`: `_get_base_query().where(CommentModel.id == comment_id)` — returns None if not found or is_deleted
- `get_by_task(task_id)`: `_get_base_query().where(CommentModel.task_id == task_id).order_by(CommentModel.created_at)`
- `create(comment)`: flush+commit, return entity with user loaded
- `update(comment_id, content)`: update content + updated_at = datetime.utcnow(); return updated entity
- `soft_delete(comment_id)`: set is_deleted=True; return True if found, False if not

`_to_entity(model)` → `Comment` domain entity. Include `user` on the entity so DTOs can map author.

```python
class ListAttachmentsUseCase:
    async def execute(self, task_id: int) -> List[AttachmentResponseDTO]

class UploadAttachmentUseCase:
    """Validates file, stores to disk, creates DB record."""
    BLOCKED_EXTENSIONS = {".exe", ".sh", ".bat", ".ps1", ".msi", ".dmg"}
    MAX_SIZE_BYTES = 25 * 1024 * 1024  # 25 MB

    async def execute(self, task_id: int, filename: str, content: bytes, uploader_id: int) -> AttachmentResponseDTO
    # 1. Check extension: Path(filename).suffix.lower() in BLOCKED_EXTENSIONS → raise ValueError("File type not allowed")
    # 2. Check size: len(content) > MAX_SIZE_BYTES → raise ValueError("File exceeds 25 MB limit")
    # 3. Generate stored name: f"{uuid.uuid4()}{suffix}"
    # 4. Write to Backend/static/uploads/tasks/{stored_name}
    # 5. Create FileModel record: file_name=original, file_path=relative, file_size=len(content)
    # 6. Return AttachmentResponseDTO

class DeleteAttachmentUseCase:
    async def execute(self, file_id: int, requester_id: int) -> None
    # Only the uploader or admin can delete
```

The upload path must be relative: `"static/uploads/tasks/{filename}"` (matching Phase 2 avatar pattern `"uploads/avatars/uuid.ext"`).

Create the uploads/tasks directory if it doesn't exist using `Path(...).mkdir(parents=True, exist_ok=True)` before writing.

**attachments.py router**:
```python
router = APIRouter()

GET  /task/{task_id}      → list_attachments(current_user=Depends(get_task_project_member))
POST /                    → upload_attachment(task_id: int = Form(...), file: UploadFile = File(...), current_user=Depends(get_task_project_member))
     → reads await file.read(), calls UploadAttachmentUseCase.execute()
     → catches ValueError → HTTPException(400) for blocked extension, HTTPException(413) for size
GET  /download/{file_id}  → download_attachment(file_id, current_user=Depends(get_current_user))
     → get file record, read bytes from disk, return FileResponse or Response with bytes
DELETE /{file_id}         → delete_attachment(file_id, current_user=Depends(get_current_user))
```

For the download endpoint: use `fastapi.responses.FileResponse` with the stored absolute path. Guard: verify file exists in DB (is_deleted=False) before serving. If file missing from disk, return 404.
