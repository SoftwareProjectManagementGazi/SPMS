`upgrade()` must do (all guarded with exists checks):
1. Create `task_dependencies` table if not exists:
   - id (Integer, PK), task_id (Integer FK tasks.id CASCADE), depends_on_id (Integer FK tasks.id CASCADE)
   - dependency_type (String(20), NOT NULL, server_default "blocks")
   - created_at (DateTime timezone, server_default now())
   - UniqueConstraint("task_id", "depends_on_id", name="uq_task_dependency")
   - NO TimestampedMixin — hard delete only (per research: dependencies have no audit value)

2. Add `task_key` (String(20), nullable) to `tasks` table if column missing
3. Add `series_id` (String(36), nullable) to `tasks` table if column missing
4. Add `task_seq` (Integer, NOT NULL, server_default "0") to `projects` table if column missing
5. Add `file_size` (Integer, nullable) to `files` table if column missing

6. Backfill existing tasks: after adding task_key column, execute:
   `UPDATE tasks SET task_key = (SELECT key FROM projects WHERE projects.id = tasks.project_id) || '-' || tasks.id WHERE task_key IS NULL`

`downgrade()` drops `task_dependencies` table, removes the 4 columns.

```python
from sqlalchemy import Column, Integer, String, DateTime, ForeignKey, UniqueConstraint
from sqlalchemy.sql import func
from sqlalchemy.orm import relationship
from app.infrastructure.database.models.base import Base

class TaskDependencyModel(Base):
    __tablename__ = "task_dependencies"
    id = Column(Integer, primary_key=True)
    task_id = Column(Integer, ForeignKey("tasks.id", ondelete="CASCADE"), nullable=False)
    depends_on_id = Column(Integer, ForeignKey("tasks.id", ondelete="CASCADE"), nullable=False)
    dependency_type = Column(String(20), nullable=False, server_default="blocks")
    created_at = Column(DateTime(timezone=True), server_default=func.now())
    __table_args__ = (UniqueConstraint("task_id", "depends_on_id", name="uq_task_dependency"),)

    task = relationship("TaskModel", foreign_keys=[task_id], backref="blocks")
    depends_on = relationship("TaskModel", foreign_keys=[depends_on_id], backref="blocked_by")
```
