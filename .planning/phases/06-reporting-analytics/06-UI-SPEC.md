## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (icon + label), badge padding |
| sm | 8px | Compact element spacing, filter chip gaps |
| md | 16px | Default card padding, between filter controls |
| lg | 24px | Section padding, between chart panels |
| xl | 32px | Layout gaps, page content gap from filter bar |
| 2xl | 48px | Major section breaks (filter bar to chart grid) |
| 3xl | 64px | Page-level top/bottom padding |

Exceptions:
- Touch targets on leaderboard sort buttons: minimum 44px height (accessibility)
- Chart panel CardContent height: 256px (h-64) — matches existing stub layout from `reports/page.tsx`
- Export button minimum width: 120px to prevent width collapse during spinner state

## Color

All color values reference the CSS custom properties defined in `Frontend/app/globals.css`. Hex approximations are provided for executor reference only.

| Role | CSS Variable | Approx Hex | Usage |
|------|-------------|------------|-------|
| Dominant (60%) | `--background` | #f4f4f8 (light) / #1a1a21 (dark) | Page background, AppShell background |
| Secondary (30%) | `--card` / `--sidebar` | #ffffff / #ffffff (light) | Chart panel cards, filter bar surface, leaderboard table |
| Accent (10%) | `--primary` | oklch(0.55 0.22 264) ≈ #4f46e5 indigo | See reserved list below |
| Destructive | `--destructive` | oklch(0.55 0.22 25) ≈ #dc2626 red | Destructive actions only — none in this phase |

**Accent reserved for:**
- Active sort column header indicator in the leaderboard table
- "Dışa Aktar" primary button background
- Primary CTA "Yeni Sprint Başlat" button in burndown empty-state overlay
- Sprint Burndown area chart stroke (`#4f46e5`) and fill (`#ede9fe`)
- Velocity Trend bar fill (`#4f46e5`)
- Filter bar active state indicators (selected project highlight ring)
- "Tüm Raporları Gör →" link text in manager dashboard leaderboard section

**Chart color palette (recharts `COLORS` array):**

Use the five CSS chart tokens defined in globals.css. Reference by token, not hardcoded hex, so dark mode adapts.

| Token | Usage |
|-------|-------|
| `--chart-1` (indigo) | Sprint Burndown area fill + stroke; Velocity Trend bars |
| `--chart-2` (teal/green) | Task Distribution: "Done" status slice; first assignee in multi-color |
| `--chart-3` (amber) | Task Distribution: "In Progress" status slice |
| `--chart-4` (purple) | Task Distribution: additional slices |
| `--chart-5` (cyan) | Task Distribution: remaining slices |

Priority color mapping for Task Distribution (Priority toggle):
- Critical: `--priority-critical` (red)
- High: `--priority-high` (amber)
- Medium: `--priority-medium` (yellow)
- Low: `--priority-low` (muted)

Status color mapping for Task Distribution (Status toggle):
- To Do: `--status-todo` (muted gray)
- In Progress: `--status-progress` (primary indigo)
- Done: `--status-done` (teal/green)

## Interaction Contracts

### Filter Bar

| Interaction | Behavior |
|-------------|----------|
| Select a project | TanStack Query refetches all 4 chart endpoints with new `project_id`. Each panel shows its own `<Skeleton>` while loading. |
| Select assignees (multi-select) | Same refetch behavior. DropdownMenuCheckboxItem with checkmarks. |
| Change date range | Same refetch behavior. Date picker updates `dateFrom`/`dateTo` filter state. |
| Filter state initialization | On mount: fetch user's most recently active project; set as default Proje selection. Date range defaults to (today − 30 days → today). |

### Export Button

| State | Visual |
|-------|--------|
| Default | `Button variant="default"` with `Download` icon: "Dışa Aktar" + chevron. Dropdown shows on click. |
| Dropdown open | "PDF olarak indir" item + "Excel olarak indir" item. No icons needed on items. |
| Generating | Dropdown closes. Button shows `Loader2 animate-spin` + "İndiriliyor..." text. `disabled` prop. Width is fixed (no layout shift). |
| Complete | File download triggers automatically. Button returns to default state. `sonner` toast: "Rapor indirildi." (success) or "Rapor oluşturulamadı. Lütfen tekrar deneyin." (error). |

### Leaderboard Table

| Interaction | Behavior |
|-------------|----------|
| Click column header | Sort asc/desc. Active sort column shows `ChevronUp`/`ChevronDown` icon in accent color. Default sort: Tamamlanan desc. |
| Click a row | Sets Assignee filter to that user. All 4 chart panels refetch. Row receives `cursor-pointer` + `hover:bg-muted/50`. |
| On-time % column | Renders inline mini progress bar. Bar width = on-time% of cell width. Bar fill: `--chart-2` (teal) for ≥ 80%, `--chart-3` (amber) for 50–79%, `--destructive` for < 50%. Shows percentage text alongside bar. |

### Task Distribution Toggle

| Interaction | Behavior |
|-------------|----------|
| Click "Durum" tab | PieChart redraws with status slice data. Tab is active (shadcn Tabs default active styling). |
| Click "Öncelik" tab | PieChart redraws with priority slice data using priority color mapping. |
| Default | "Durum" tab is active on mount. |
