# Phase 12 — Manual UAT Checklist

**Purpose:** End-to-end verification of all 14 LIFE-* + EDIT-* requirements per SPEC Acceptance Criteria + VALIDATION.md Manual-Only Verifications table.

**How to run:** Open the application against a running Frontend2 dev server (`cd Frontend2 && npm run dev`) plus a running Backend (`cd Backend && uvicorn app.main:app --reload`), follow each row's click-through steps, observe the expected behavior, and mark Pass/Fail. Sign off when all 14 (or 15) rows pass.

**Phase 12 deliverables under test:**

- LifecycleTab with 4 sub-tabs (Overview / Milestones / History / Artifacts)
- Phase Gate inline expand with 5-error matrix + override flow
- Settings > Yaşam Döngüsü criteria editor + `enable_phase_assignment` toggle
- Settings > General methodology read-only field + Tooltip
- `/workflow-editor?projectId=X` page with 1024px viewport gate
- React Flow canvas + DnD + 4-way edge create + 5 group entry points + cloud morph
- Edge types (flow / verification / feedback) with custom dasharrays
- All-gate edge + bidirectional pair-wise semantic
- Cycle counter ×N badge (visible when count ≥ 2)
- Parallel actives in flexible / sequential-flexible
- Sequential-flexible mode with feedback transitions
- Incremental / Evolutionary / RAD presets via Şablon Yükle menu
- Editor save flow with 200/422/409/429/network error matrix
- Dirty-save guard (beforeunload + safePush DirtySaveDialog)
- Backend WorkflowEdge.bidirectional + is_all_gate fields (additive Pydantic, no migration)
- Timeline tab Gantt with milestone flag lines + popover
- PDF download from Evaluation Report
