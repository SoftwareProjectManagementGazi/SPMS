<!-- Conversion pattern from PATTERNS.md (universal for all 16 primitives): -->

File header:
```typescript
"use client"
import * as React from "react"
import { cn } from "@/lib/utils"
```

Props interface (always include className and style):
```typescript
interface ComponentNameProps {
  // all props from prototype
  className?: string
  style?: React.CSSProperties
}
```

Named export:
```typescript
export function ComponentName({ prop1, prop2, className }: ComponentNameProps) {
  return (...)
}
```

Conversion rules:
- Static layout styles -> Tailwind classes
- Token-based colors -> `bg-[var(--token)]` or `text-[var(--token)]`
- Dynamic values (depend on props) -> keep as style={}
- color-mix() expressions -> keep as style={} (Pitfall 3)
- Inset shadow tokens -> keep as style={{ boxShadow: "var(--inset-*)" }}
</interfaces>

**avatar.tsx (per D-01, D-03, D-04):**
Port from `New_Frontend/src/primitives.jsx` Avatar component:
- Props: `user: { name: string; avColor?: number; avatar?: string }`, `size?: number` (default 32), `ring?: boolean`, `className?: string`
- Renders a `<div>` with rounded-full, size set via style (dynamic), background `var(--av-${user.avColor || 1})`
- If `user.avatar` exists, render `<img>` with cover fit

- `ring` prop adds a 2px solid white border
- Use `cn()` for className merging
- Named export: `export function Avatar`

- Props: `users: Array<{ name: string; avColor?: number; avatar?: string }>`, `max?: number` (default 3), `size?: number` (default 28), `className?: string`

- If users.length > max, shows `+{remaining}` overflow badge with `var(--surface-2)` background and `var(--fg-muted)` text
- Named export: `export function AvatarStack`
