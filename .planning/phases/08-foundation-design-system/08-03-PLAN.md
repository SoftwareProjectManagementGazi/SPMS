- Icon slot: colored with tone color, flex-shrink-0
- Content area: `flex-1 text-[13px]`
- Action slot: rendered at the right end
- Default icons per tone (from lucide-react): warning = AlertTriangle, danger = AlertTriangle, success = CheckCircle, info = Info. Only use default if `icon` prop is not provided.
- Named export: `export function AlertBanner`

**toggle.tsx (per D-03, D-04):**
Port from primitives.jsx Toggle:
- Props: `on: boolean`, `onChange?: (value: boolean) => void`, `size?: "sm" | "md"` (default "md"), `className?: string`
- Size map: sm (width 32px, height 18px, knob 14px), md (width 40px, height 22px, knob 18px)
- Track: rounded-full div, background transitions between `var(--border-strong)` (off) and `var(--primary)` (on)
- Knob: absolute positioned circle, white background, transitions left position via `transition-all duration-200`
- Off position: knob at left (2px offset). On position: knob at right (track width - knob size - 2px)
- Use inline style for dynamic positioning. Use Tailwind for static: `relative cursor-pointer rounded-full`
- Click handler calls `onChange?.(!on)`
- Named export: `export function Toggle`
